GitHub Repo Gallery

The GitHub Repo Gallery pulls user data from a named GitHub account and displays profile information and lists of repos using JavaScript. Users are able to search through the repos, select a given repo, and view data on that repo, including languages used and a link to the repo on GitHub.

Guess the Word

Guess the Word is a game that pulls a random word from a list of imported words, displays a placeholder, and allows users to guess letters in the random word, updating the placeholder if they guess a letter correctly. If they guess all the letters in the word before they run out of guesses, they win!

Skills: JavaScript

Tools: Code Sandbox, JSON, GitHub Pages

Live Website
